## Problem

The ApplicationSet UI (list tile, table row, detail panel) exposes **no Refresh action**. Operators who push a new directory under a Git generator path must force reconcile via kubectl:

```bash
kubectl annotate applicationset <name> \
  argocd.argoproj.io/application-set-refresh=true --overwrite
```

The controller **already** handles this annotation тАФ Git generator bypasses revision cache when `RefreshRequired()` is true. The gap is API + UI only.

---

## Root cause (investigation)

| Layer | Status |
|-------|--------|
| Controller | `RefreshRequired()` in `pkg/apis/application/v1alpha1/applicationset_types.go` |
| Annotation | `common.AnnotationApplicationSetRefresh` = `argocd.argoproj.io/application-set-refresh` |
| Git generator | `noRevisionCache := appSet.RefreshRequired()` in `applicationset/generators/git.go` |
| Controller cleanup | Deletes annotation after reconcile in `applicationset_controller.go` |
| Webhook | Already patches same annotation in `applicationset/webhook/webhook.go` |
| **API** | **Missing** тАФ no `Refresh` RPC in `server/applicationset/applicationset.proto` |
| **UI** | **Missing** тАФ AppSet tiles/rows/details have no refresh handler |

Application refresh pattern to mirror:

- Server: `application.go` handles refresh via `Get` query param
- UI: `applications-list.tsx` тЖТ `refreshApp()` тЖТ `applications-service.get(..., refresh)`
- AppSet UI lives under `ui/src/app/applications/components/applications-list/appset-*.tsx`

---

## Proposed fix

1. **Proto** тАФ add `Refresh` RPC + request message with `name`, `appsetNamespace`
2. **Server** тАФ `Server.Refresh` in `server/applicationset/applicationset.go`:
   - Load ApplicationSet (respect namespaced AppSets)
   - Patch `argocd.argoproj.io/application-set-refresh: "true"`
   - Enforce RBAC `applicationsets, update`
3. **Codegen** тАФ `make codegen` (in isolated env)
4. **UI service** тАФ `applications-service.ts` тЖТ `refreshApplicationSet(name, namespace)`
5. **UI components** тАФ Refresh action on:
   - `appset-tile.tsx`
   - `appset-table-row.tsx`
   - `appset-resource-details.tsx`
   - Mirror Application refresh UX (disabled while refreshing)
6. **Tests** тАФ Go unit tests for `Server.Refresh` (annotation patch, RBAC, not-found)
